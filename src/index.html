Airstrip.js

Third-party libraries manager (a.k.a. airstrip)

Abstract

This domain host a build result of airstrip, meant to provide commonly used (third-party) frameworks and js libraries. They are usually provided in two versions - release and trunk. Though you can link directly to them, it's recommended you use a jsboot stack instead, or deploy airstrip for yourself. Note that some of these libraries are patched, or even forked.
